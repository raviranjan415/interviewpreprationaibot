# AI Interview Practice Assistant

An interactive chatbot designed to help users practice technical interview questions, featuring a modern UI with Tailwind CSS and realistic typing effects.

## Features

- **Technical Focus**: Specialized in programming, data structures, algorithms, and system design topics
- **Modern UI**: Built with Tailwind CSS for a clean, responsive design
- **Interactive Elements**: Z-index effects, hover animations, and realistic typing
- **Dark/Light Mode**: Toggle between themes with persistent preference
- **Topic Selection**: Quick access to common interview topics
- **Code Highlighting**: Proper formatting for code examples
- **DSA Implementation**: Uses data structures like Queue, Stack, and Trie in the codebase

## Tech Stack

- **Frontend**: HTML, JavaScript, Tailwind CSS
- **Backend**: Python, Flask, Gemini API
- **Data Structures**: PriorityQueue, Stack, Trie, RequestQueue
